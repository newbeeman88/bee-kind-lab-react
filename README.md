## About the frontend project
The project is hosted on [Vercel](https://vercel.com/), which provides the developer tools and cloud infrastructure to build, scale, and secure a faster, more personalized web.

Visit https://bee-kind-lab-1111.vercel.app/ for a quick tour!

## Features

### Core Features
- **Posts & Photography**: Share your beekeeping experiences, beautiful photos, ask questions, and help others
- **Interactive Feed**: Like, comment, and engage with fellow beekeepers' posts
- **User Profiles**: Customize your profile with bio, location, and experience level
- **Communities**: Join or create local beekeeping communities
- **Events**: Organize and attend beekeeping events and workshops
- **AI Assistant**: Get beekeeping advice from our intelligent chatbot (logged-in users only / Under deployment)

### Design & UX
- **Beautiful UI**: Warm, natural design with honey gold and forest green themes
- **Responsive**: Works perfectly on desktop, tablet, and mobile devices (still working on it)
- **Dark/Light Mode**: Toggle between light and dark themes
- **Smooth Animations**: Delightful micro-interactions throughout the app
- **Accessibility**: Built with accessibility best practices
- **Multiple Color Themes**: Choose from 6 beautiful color schemes
  - **Honey Gold** (default) - Warm honey and green tones
  - **Ocean Breeze** - Cool blues and teals
  - **Lavender Dream** - Purple and pink gradients
  - **Forest Green** - Deep greens and earth tones
  - **Sunset Orange** - Warm oranges and yellows
  - **Deep Purple** - Rich purples and magentas

### Technical Solutions
- **Modern Stack**: React 18, TypeScript, Vite, Tailwind CSS v4
- **Component Library**: Custom ShadCN/UI components
- **Image Handling**: Unsplash integration for beautiful beekeeping images
- **State Management**: React hooks and local state
- **Performance**: Optimized loading, lazy loading, and efficient rendering

## How to Start

### Prerequisites
- Node.js 18+
- npm, yarn, or pnpm

### Installation

1. **Clone the repository**

```bash
$ git clone https://github.com/newbeeman88/bee-kind-lab-react.git

$ cd bee-kind-lab-react
```

2. **Install dependencies**

```bash
$ npm install
# or
$ yarn install
# or
$ pnpm install
```

3. **Start the development server**

```bash
$ npm run dev
# or
$ yarn dev
# or
$ pnpm dev
```

4. **Access the website**

Open your browser and navigate to `http://localhost:3000`
